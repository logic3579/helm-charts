Helm Charts Repository

Welcome to the Helm Charts repository. This repository contains a collection of Helm charts that can be used to deploy applications on Kubernetes.

Quick Start

Add this repository to Helm:

helm repo add my-repo https://logic3579.github.io/helm-charts
helm repo update

Available Charts

Chart Name Version Description Install Command
mychart Latest A sample Helm chart helm install mychart my-repo/mychart
mylibchart Latest A library chart for common templates helm dependency update ./my-chart

Documentation

Helm must be installed to use the charts. Please refer to Helm's documentation to get started.

Build a Helm package

To create a helm chart:

helm create charts/my-chart

Check configuration for all charts:

helm lint charts/*

Package all helm chart into chart archive:

helm package charts/*

Install a Chart

Once Helm has been set up correctly, you can install a chart:

helm install my-chart my-repo/my-chart

To uninstall the chart:

helm uninstall my-chart
